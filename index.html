Промпт для карточек

Готовый промпт для создания интерактивных карточек с помощью нейросети.

Промпт

При помощи данного промпта у вас получатся яркие, интерактивные карточки, которые вы сможете использовать на своих уроках без интернета.

Создай HTML-файл с переворачивающимися карточками (flip cards). Требования: Сделай 10 карточек по теме животные для 4 класса У каждой карточки две стороны: лицевая — слово на английском + картинка я вставлю ее из локальной папки с компьютера в формате images/card1.jpg обратная — перевод на русский Сделай современный, аккуратный дизайн (скруглённые углы, тени, приятные цвета, эффект наведения hover ) Карточка должна быть на экране одна и занимать 50 процентов сайта , после того как ученик ее изучил он может перейти к другой нажав кнопку следующая весь сайт с английским интерфейсом добавь также фон это будет изображение с прозрачностью 70 % чтобы не отвлекать внимание от карточек, он также будет с локальной папки images/background.jpg
